Name: SCR: longboat sail mesh fix
Version: 1.0
Date: 5/01/2011
Category: Models and Textures
Author: Alaisiagae

Description
===========
Contains the longboat01 mesh with fixed/improved UV mapping for the green sail.


Details
=======
Contains the longboat01 mesh with fixed/improved UV mapping for the green sail. Pluginless replacer. Will affect all longboat01 meshes in-game.


Install
=======
1. Extract the files to a temporary location.
2. Examine the folder structure and make corrections where necessary.
3. Copy files to (install folder)\Morrowind\Data Files\
4. Start Morrowind Launcher and play.


Uninstall
=========
To remove this mod, simply delete the following:

Meshes\x\Ex_longboat01.nif


Upgrade
=======
1. Remove the previous version's textures.
2. Add the new version's textures.
3. Start Morrowind and play. There is no effect on saved games and shouldn't corrupt any saves.


Incompatibility
===============
This mod will conflict with another mod that modifies the exact same mesh. It will not conflict with pluginless texture replacers. Will affect all longboat01 meshes placed in-game.


Known Issues or Bugs
====================
None that I am aware of. If something is wrong, let me know.
